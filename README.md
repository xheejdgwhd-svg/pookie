# Cookie Stock — Лэндинг (статичный)

Готовая верстка под макет из скрина. Без зависимостей: чистый HTML/CSS/JS.
- Секции: шапка, hero, кнопки, карусель категорий, плавающая корзина.
- Карусель скроллится стрелками и свайпом.
- Цвета/тени/градиенты приближены к макету.
- Шрифты: Inter через Google Fonts.

## Запуск
Откройте `index.html` в браузере.

## Настройка
- Тексты и категории правятся прямо в `index.html`.
- Цвета и темing в `styles.css` (переменные в `:root`).
